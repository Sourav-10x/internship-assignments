1. What is Git

Git is a distributed version control system used to track changes in files and source code.

2. Why Git is useful

Git helps developers track changes, collaborate with teams, manage versions, and restore previous versions of files.

3. Difference between Git and GitHub

Git is a version control tool used locally.
GitHub is a cloud platform used to store and share Git repositories online.

Three git commands learned:
git add -Stages file for commit
git commit -Saves changes to repository
git status -Shows repository status
